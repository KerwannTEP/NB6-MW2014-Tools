# NB6-MW2014-Tools
Tools related to running NBODY6++GPU with MWPotential2014. 

## Description

This repository gathers tools to compile and run `NBODY6++GPU` (available [here](https://github.com/KerwannTEP/Nbody6ppGPU)) subject to the Milky-Way potential model `MWPotential2014` ([Bovy 2015](https://ui.adsabs.harvard.edu/abs/2015ApJS..216...29B/abstract)). It also provides tools to read the binary outputs and plot the cluster evolution in the galactico-centric frame.

Its primary purpose is to serve as comparison with KRIOS (available [here](https://github.com/UNC-Dynamics/KRIOS) soon) runs described in `Cook & al (2025)`, but it may be of use to everyone wishing to study stellar streams.
